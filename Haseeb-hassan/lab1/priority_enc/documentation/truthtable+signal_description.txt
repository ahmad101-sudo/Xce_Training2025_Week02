signal description
---------------------------------------------------------------------------------------------------
Inputs:

	enable: Control input (active high).

	data_in[7:0]: Input lines (bit7 = highest priority).

Outputs:

	encoded_out[2:0]: Binary code of highest-priority active input.

	valid: Indicates at least one input is active (when enable=1).

Internal logic:

	If enable=0, block is disabled.

	If enable=1, encoder scans from data_in[7] to data_in[0] and outputs the position of the first 1.

	valid=1 if any bit is set, else 0.
